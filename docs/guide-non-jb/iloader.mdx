iloader permette di eseguire il sideload di qualsiasi app su iOS/iPadOS facilmente.

Integra inoltre il supporto nativo a [SideStore](sidestore) e permette di importare con un click il pairing file.

## Come utilizzarlo

1. Installa usbmuxd sul tuo PC:
    * macOS: installato di default
    * Windows: scarica iTunes ([MS-Store](https://apple.co/ms) o [.exe](https://itunes.com/download))
    * Linux: potrebbe essere installato di default, altrimenti installalo col tuo package manager (es. `sudo apt install usbmuxd`)
2. Installa [iloader](https://github.com/nab138/iloader/releases/latest)
3. Collega il tuo dispositivo al computer
4. Accedi col tuo ID Apple
5. Seleziona l'azione desiderata
